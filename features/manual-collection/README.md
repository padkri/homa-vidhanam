# Manual Collection

**Status:** Done
**Spec:** #4
**Priority:** P2

## Overview
Build a comprehensive collection of homam manuals covering major Hindu deities and rituals.

## Scope
- Individual homam manuals as JSON data files in `src/data/`
- Each manual includes sections, slokas, instructions, and diagrams
- Configured via `src/data/config.json`

## Current Manuals (17)
Siva, Ganesh, Vishnu, Durga, Chandi, Dattatreya, Hanuman, Kalabhairava, Kali, Kartikeya, Lalitha, Mahalakshmi, Narasimha, Saraswati, Srirama, Tara, Tarpanam
