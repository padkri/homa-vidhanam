# Multi-language Support

**Status:** Done
**Spec:** #3
**Priority:** P1

## Overview
Support displaying homam manuals in multiple languages — English, Telugu, and Hindi/Devanagari — with a language selector in the UI.

## Scope
- English (Latin script) display
- Telugu script display
- Hindi/Devanagari script display
- Language selector in top-right of UI
- All manual JSON files include translations for each supported language

## Key Decisions
- Three languages supported: English, Telugu, Hindi
- Slokas stored in English (Roman transliteration), Telugu, and Devanagari scripts
